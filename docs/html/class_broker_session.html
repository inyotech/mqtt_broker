void BrokerSession::packet_received ( std::unique_ptr< Packet >  packet)
overridevirtual

PacketManager callback.

Invoked by the compsed PacketManager when it receives a complete control packet. The default handler methods will be passes a reference to this packet. Memory occupied by the packet will be freed by standard C++ std::unique_ptr rules. It is the responsibility of subclasses to maintain the packet lifetime.

Parameters
packetReference counted pointer to a packet.
