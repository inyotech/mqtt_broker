Detailed Description

Deserialization class.

Methods are provided to read native types from the wire encoded control packets received over the network connection. MQTT 3.1.1 standard decoding methods are implemented.

Constructor & Destructor Documentation

PacketDataReader::PacketDataReader ( const packet_data_t &  packet_data)
inline

Constructor

Accepts a reference to a packet_data_t container that contains data received directly over a network connection. The class also contains a current offset pointer that is initialized to point to the beginning of the container. Each data read will advance the offset pointer forward to the next item in the container.
