MQTT Broker and Clients Documentation

The MQTT publish/subscribe protocol is a simple lightweight messaging protocol for distributed network connected devices.

About

Open source, asynchronous, C++ MQ Telemetry Transport client/server implementing the 3.1.1 OASIS standard available here.

Installation

Requirements

This MQTT implementation is completely asynchronous and requires the Libevent networking library. Other than that there are no other external run-time dependencies.

  • A C++11 conformant compiler.
  • Libevent
  • CMake

Building

  1. Clone this repository.
    1 $ git clone https://github.com/inyotech/mqtt_broker.git
    2 $ cd mqtt_broker
  2. Create a build directory.
    1 $ mkdir build
    2 $ cd build
  3. Generate build files.
    1 $ cmake ..
  4. Build/
    1 $ make

Example

  • Open a terminal and execute the broker.
    1 $ mqtt_broker
  • In a second terminal execute a subscriber.
    1 $ mqtt_client_sub --topic 'a/b/c'
  • Execute a publisher in a third terminal.
    1 $ mqtt_client_pub --topic 'a/b/c' --message 'published message'
